# Scheduler AI

Offline AI chatbot buat scheduling & reminder. Ngerti bahasa Indonesia & Inggris otomatis.

## Setup

### 1. Install Flutter
Download di https://flutter.dev/docs/get-started/install

### 2. Install dependencies
```bash
cd scheduler_ai
flutter pub get
```

### 3. Run
```bash
# Android / iOS
flutter run

# Windows
flutter run -d windows

# macOS
flutter run -d macos

# Linux
flutter run -d linux
```

## Cara pakai

Tinggal ketik natural aja:
- `"Besok jam 3 ada meeting"`
- `"Tanggal 20 jam 9 pagi dokter gigi"`
- `"Jan 20 at 9am dentist"`
- `"Tomorrow at 3pm team standup"`

Bot otomatis detect bahasa lo dan balas sesuai.
